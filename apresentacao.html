Apresentação

    Para construção do site em questão foi utilizada a Digimon API, está podem ser acessada em https://digimon-api.vercel.app/.
    Desta foram consumidas as chamadas GET https://digimon-api.vercel.app/api/digimon e GET https://digimon-api.vercel.app/api/digimon/level/:level
    Para disponibilização da busca de digimons utilizamos a chamada GET https://digimon-api.vercel.app/api/digimon/name/:name junto ao metodo fetch e a função await, para que o processo aguardasse o input do nome para trazer o resultado da API, este foi conectado ao html e utiliza o metodo Find para localizar o Digimon especificado dentro do array pelo seu nome e posteriormente acrescentar seus dados em tela, com o metodo innerHTML.
    Já para a listagem de digimons utilizamos a chamada GET https://digimon-api.vercel.app/api/digimon/level/:level, que acionada pelo metodo fetch, devolve um array de Digimons, esse array é salvo dentro de uma constante como JSON e é percorrido utilizando a função Reduce, onde cada um dos elementos recebe um acrescimo de um código HTML que posteriormente é levado a tela utilizando o innerHTML.
